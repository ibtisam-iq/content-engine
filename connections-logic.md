# LinkedIn Connection & Messaging Logic — Notes by Muhammad Ibtisam Iqbal

> **Purpose:**  
> To understand how LinkedIn connection levels, visibility modes, Premium settings, and the "Message / Connect" buttons actually work.

---

## 🌐 1. Private Mode vs Visible Mode

- **Private Mode** and **Visible Mode** are independent of having Premium.  
- Any user (Free or Premium) can switch between them anytime.  
- **Visible Mode:** the person you view can see your name & headline.  
- **Private Mode:** you appear as *Anonymous LinkedIn Member* — no trace left.

> Premium only affects *analytics depth*, not *visibility control.*

---

## 💬 2. Open Message Feature

- **"Open Message"** (also called *Open Profile*) is a **Premium-only feature.**  
- When enabled, anyone can send you a message directly — even if not connected.  
- Free users **cannot** enable or receive open messages; they rely on “Connect” requests.  
- Recruiters and HR professionals often keep this **enabled** to attract talent.

---

## 🧩 3. “Message” vs “Connect” Button Logic

| Case | Account Type | Open Message Status | Button Shown |
|------|----------------|--------------------|---------------|
| Free User | — | (Feature not available) | ➕ Connect |
| Premium User | Enabled | 💬 Message |
| Premium User | Disabled | ➕ Connect |
| 1st-Degree Connection | Any | (Always active) | 💬 Message |

**Interpretation:**
- Seeing **“Message”** on a 2nd-degree profile → that person is Premium **and** has Open Messages ON.  
- Seeing **“Connect”** on a 2nd-degree profile → either Free user, or Premium with Open Messages OFF.  
- 1st-degree connections always show **“Message”**, regardless of plan.

---

## 🧭 4. Connection Degree Meaning

| Degree | Meaning | Communication |
|---------|----------|----------------|
| **1st** | Already connected | Direct message possible |
| **2nd** | Share mutual connections | Can “Connect” or message if Open Profile is ON |
| **3rd** | No mutual connection | “Follow” or InMail only (Premium required) |

---

## ❤️ 5. Summary

- Visibility = controlled by **Private Mode**, not Premium.  
- Premium adds **Open Message** and **InMail**, not invisibility.  
- “Message” or “Connect” button changes based on **relationship + message permission.**  
- Recruiters often use Premium with Open Messages enabled — allowing candidates to contact them easily.  
- Free users rely on the **Connect** request system only.
